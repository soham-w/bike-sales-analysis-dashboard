# bike-sales-analysis-dashboard

Project Overview

This project presents an interactive Bike Sales Dashboard built using Power BI to analyze sales performance across different regions, customers, and product categories.

The dashboard enables users to explore trends, identify top-performing segments, and make data-driven business decisions.

Dataset Information

Dataset Name: Bike Sales Data
Records: 60,000+ rows
Fields Included:
Region, Country
Customer & Business Segment
Product Category & Model
Sales Date
List Price & Unit Price
Order Quantity
