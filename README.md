# bike-sales-analysis-dashboard

Project Overview

This project presents an interactive Bike Sales Dashboard built using Power BI to analyze sales performance across different regions, customers, and product categories.

The dashboard enables users to explore trends, identify top-performing segments, and make data-driven business decisions.

## Dataset 
- <a href = "https://github.com/soham-w/bike-sales-analysis-dashboard/blob/main/Bike_Data.xlsx">bike-sales</a>

## Key Insights
The dashboard provides valuable insights such as:
 - Region-wise and country-wise sales analysis
 - Sales trends over time
 - Top-selling bike categories and models
 - Customer segmentation insights
 - Revenue and quantity analysis

## Features
 - Interactive filters and slicers
 - Dynamic charts and visuals
 - KPI cards for quick insights
 - Clean and user-friendly dashboard

## Tools & Technologies
 - Power BI
 - Microsoft Excel
 - Data Cleaning & Transformation
 - DAX (Data Analysis Expressions
